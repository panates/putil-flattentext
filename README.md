Simple utility that converts manipulate \t \b \r characters and flattens text


## Installation

  - `npm install putil-flattentext --save`

## Usage

`flattentext(str, options)`


```javascript
const flattenText = require('./');
let text =
    'aaa\n'+
    '\tbbb\n'+
    '\t\tccc\n'+
    '\bddd\r'+
    'eee';
console.log(flattenText(text));
```
Result output
```
aaa
  bbb
      ccc
    ddd
eee
```

  - node `>= 6.x`;
